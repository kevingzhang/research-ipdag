# Gist
Research on IPFS DAG
## Run
npm i 

Open terminal 1, run npm start
Open terminal 2, run npm run cli

In terminal 2, you can see prompt like ALOHA>
type command as below
- do getPeers
- do hasPeer <peerId>
- do leave
- do sendTo <peerId> <msg>
- do broadcast <msg>
- do get
- do add

Terminal 1 will receve those command and output result.
